# Diwali-Sales-Analysis-Python-

**Diwali Sales Analysis using Python**
**Overview**

This project analyzes Diwali sales data to understand customer purchasing behavior and identify top-performing products and categories using Python.

**Objectives**
- Analyze customer demographics and buying patterns
- Identify high-revenue product categories
- Extract insights to improve festive-season sales strategies

**Tools Used**
- Python
- Pandas, NumP
- Matplotlib, Seaborn
- Jupyter Notebook
